O que é o HTML

O HTML é uma linguagem de marcação padrão para páginas WEB, criado pelo Tim Berners-Lee no ano de 1991. O HTML está presente em praticamente todos os sites da internet.

A linguagem de marcação HTML utiliza de tags que indicam seu propósito e relação com outros elementos na página, facilitando a criação de documentos estruturados

Lembrando pessoal, HTML não é uma linguagem de programação, logo não podemos dizer: “Eu programo em HTML”.

O que são atributos

O HTML possui uma série de atributos que podem ser adicionados a esses elementos para modificar seu comportamento ou aparência, como classes, IDs, e atributos de estilo. Essa flexibilidade permite aos desenvolvedores web criar layouts complexos e interativos que funcionam de maneira consistente através de diferentes dispositivos e navegadores.

Temos diversos atributos na linguagem de marcação HTML, como por exemplo definir o tipo de dado que será inserido em um campo...
